Project: Deduction Game

Technologies: HTML, CSS, JavaScript, jQuery

A single-player memory concentration game to help freelance workers become more familiar with self-employed tax benefits. Two tiles are evaluated for each round of play. If the two tiles chosen are a match, the matched tiles are cleared from the game board and the counter for correct guesses increases by one. If the two tiles chosen are a mismatch, the counter increases by one for incorrectly guessed matches and the tiles return to their hidden state until they are matched correctly.

Project: TechAssess

Technologies: HTML, CSS, Ruby, Ruby on Rails, ERB

An assessment tool that prompts users to answer questions about their computer's health as well as how often maintenance steps are performed. A risk level score is provided to the user based on answers to the questions. Articles are also provided to help each user learn about topics such as safe browsing habits and operating system updates. The goal is to help users understand what could be done on a routine basis to prevent catastrophic events such as data loss, mechanical failure, or identity theft.

Group Project: MenuBuilder

Technologies: HTML, CSS, Node.js, Express, JavaScript, jQuery, API Integration

Collaborative group project built to help users view nutrition information prior to ordering a meal. Searching for a food item returns a list of menu items from the Nutritionix API. As the user selects individual menu items, a calculator updates in real time to display the meal's total nutritional value. The next phase of implementation will be to implement user authentication (using Passport.js and OAuth) and the ability to display previously saved meals. The site is optimized for desktop and mobile use.

Project: Job Trackr

Technologies: HTML, CSS, Ruby, Ruby on Rails, ERB, JavaScript, jQuery

A responsive site that organizes an applicant's job search by storing data associated with the application process in one location. Users sign in and log application information after they've applied for a job (such as the date applied, a link to the job post, and a link to the resume and cover letter version submitted). Each application record can be viewed separately or in a summary view of all applications. Users can add notes to an application, delete an application, and add an application as needed.
